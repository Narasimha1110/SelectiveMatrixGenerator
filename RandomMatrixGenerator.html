Operational instructions

This web page generates rectangular matrices in the range of entered value


Please enter values and click on generate as may times you want, It will generate the submatrices within the range of entered value.

For example if you enter value as 2, It will generate random two sub matrices with in 20x16 matrix on continuous clicking on generate button
